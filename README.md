# Sanctum - My private place for doing creative work

## Introduction
My neovim config, built around the concept of minimalism and coziness. Just like my hyprland/niri and other system configs.
The goal is to create a neovim config which is useful and minimal instead of flashy and feature-heavy.

This neovim config is built for mainly the following frameworks / programming languages:
- Laravel / PHP
- React / JS / TS
- Python
- Go
